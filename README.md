The application uses linked object classes to construct a complete project 
management system, preserving connections between tasks, projects, team members, 
vendors, and clients. Following an object-oriented design paradigm, each entity class 
maintains references to linked objects while containing its own data and activities. 
The system organizes functionality through a hierarchical menu interface that guides 
users through creation, update, and deletion procedures while maintaining all object 
relationships. Core activities include project-task hierarchies, vendor and human 
resource allocation, progress monitoring via status updates, and comprehensive views 
of all interrelated entities
